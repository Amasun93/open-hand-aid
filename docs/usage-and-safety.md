# 使用与安全边界

OpenHandAid 当前仍在迭代，不属于医疗器械，不替代医生、康复师或专业治疗建议。

## 可能适合尝试

- 手指自然张开困难。
- 抓握稳定性不足。
- 日常轻量夹取动作受限。
- 有明确动作目标，并愿意反馈佩戴体验。

## 暂不建议自行尝试

- 皮肤破损。
- 明显疼痛、麻木或持续勒痕。
- 严重痉挛。
- 关节畸形固定。
- 近期手部手术。
- 医生或康复师明确不建议佩戴外部辅具。

## 使用时请注意

- 首次佩戴建议有人协助。
- 不要长时间连续佩戴，具体时长仍待测试确认。
- 如出现疼痛、麻木、勒痕或不适，应立即停止使用。
- 不要把单次有效体验理解为普遍疗效。
- 不同病种、手型、肌张力和疼痛敏感度差异很大。
