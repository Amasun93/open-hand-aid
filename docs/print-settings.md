# 打印参数

当前参数来自 MakerWorld 页面与项目素材整理。

## 推荐配置

| 参数 | 当前值 |
| --- | --- |
| 层高 | 0.2 mm |
| 墙数 | 2 |
| 填充 | 15% |
| 估算打印时间 | 约 2.3 小时 |

## 材料

| 材料 | 建议用途 | 状态 |
| --- | --- | --- |
| PETG | 关节、骨架、支撑类部件 | `finger.stl`、`strap-holder-v2.stl`、`strap-holder-thumb-v2.stl` 暂按 PETG 处理，后续可继续校准 |
| TPU95A | 手背/掌部柔性固定结构 | `hand-heated-part.stl` |
